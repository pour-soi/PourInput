# Code of Conduct

Mouser Multi-Action should be a welcoming project for users, testers, and contributors.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent when asking for clarification.
- Keep feedback focused on the work, not the person.
- Be patient with hardware-specific debugging. HID++ behavior can vary by device, receiver, firmware, operating system, and installed Logitech software.
- Share enough context for others to reproduce your issue.

## Unacceptable Behavior

- Harassment, insults, threats, or personal attacks.
- Discriminatory language or behavior.
- Posting private information without permission.
- Repeatedly derailing issues or pull requests.
- Demanding unpaid support from maintainers or contributors.

## Enforcement

Maintainers may edit, hide, or remove comments that violate this code of conduct. Repeated or severe violations may result in being blocked from participating in the project.

If you need to report a conduct concern, contact the maintainer through the repository's available GitHub contact path.
